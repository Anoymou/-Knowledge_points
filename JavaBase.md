**数据类型、运算符、键盘录入**

**数据类型**

Java中的数据类型有两大种，分别是 基本数据类型 、引用数据类型。

------

**基本数据类型**

​	整型：`byte、short、int、long`

​		` byte`：占一个字节、数据范围 ：-128~127

​		 `short`：占两个字节、数据范围 ：-32768~32767

​		 `int`：占四个字节、数据范围 ：-2^31~2^31 -1

​		 `long`：占八个字节、数据范围：-2^63~2^63 -1 书写的时候在后面加L

​	浮点型：`float、doouble`

​		 `float`：占四个字节、数据范围：约 ±3.4 × 10^38  书写的时候在后面加f

​		 `double`：占八个字节、数据范围：约 ±1.7 × 10^308

​	字符型：`char`：占两个字节、数据范围：0~2^16

​	布尔型：boolean：占一个字节、只能存储`true/false`

数据类型转换

​	自动类型转换：数据范围小的变量可以直接赋值给数据范围大的变量，本质是在较小数据类型前面，补了若干个字节 

​	表达式的自动类型转换：多种数据类型参与运算，其结果以大的数据类型为准

​	强制转换：目标数据类型  变量名  =  (目标数据类型)被转换的数据;

------

**引用数据类型**

​	存储的是对象的地址（引用），存储在堆内存

​	数组、类、接口、枚举、`String` 也是引用类型，但它是不可变对象（Immutable Object）

Java为每种基本数据类型都提供了一个对应的包装类，用于处理对象操作

| 基本类型  |   包装类    |
| :-------: | :---------: |
|  `byte`   |   `Byte`    |
|  `short`  |   `Short`   |
|   `int`   |  `Integer`  |
|  `long`   |   `Long`    |
|  `float`  |   `Float`   |
| `double`  |  `Double`   |
|  `char`   | `Character` |
| `boolean` |  `Boolean`  |

这里我们介绍几个特殊数据类型

​	`var`关键字：允许根据初始值推断类型，但必须在方法内部使用

​	一般应用在 一个变量后面 .var idea会自动判断数据类型 并在左边加上数据类型和变量名。

​	`null`只能用于引用类型，表示对象引用为空。	

```
   String str = null;  // str 不指向任何对象
```

------

**运算符**

算术运算符、赋值运算符、自增自减运算符、比较运算符、逻辑运算符、三元运算符

**算术运算符**

​	+、-、*、/、%(取余)

​	整数除法：`int / int` 结果是 `int`，小数部分被截断。

​    小数计算：`double / int` 结果是 `double`。

**赋值运算符**

​	=、+=、-=、*=、/=、%=

​	需要注意的+=如果左边是字符串的时候，起拼接字符串的作用

**自增、自减运算符（++、--）**

​	用于变量递增或递减1

​	放在变量前面的时候 先执行自增自减操作 反之 后执行自增自减操作

**比较运算符**

​	\>、>=、<、<=、==、!=

**逻辑运算符**

​	&&、||、!

​	&& 与运算符 同真才为真 

​	|| 或运算符 一真则为真

短路逻辑

​	&& 遇到 **false** 直接返回，不执行后续条件。

​	|| 遇到 **true** 直接返回，不执行后续条件。

**三元运算符**

​	用于简化if-else语句

​	语法：条件表达式?值1:值2

**键盘录入**

​	`Scanner`是Java常用获取键盘输入的方式。

​	导入`Scanner`类

```
import java.util.Scanner;
```

​	创建`Scanner`对象

```
Scanner sc = new Scanner(System.in);
```

​	使用`Scanner`获取用户输入

​	`next()`：读取一个单词（遇到空格、回车停止）

​	`nextLine()`：读取一整行字符串（包括空格、遇到回车停止）

​	`nextInt()`：读取整数

​	`nextDouble()`：读取双精度浮点数

​	`nextFloat()`：读取单精度浮点数

​	`nextBoolean()`：读取布尔值（true/false）

​	`nextLong()`：读取长整数

​	`nextByte()`：读取字节型数据

​	`nextShort()`：读取段整数
